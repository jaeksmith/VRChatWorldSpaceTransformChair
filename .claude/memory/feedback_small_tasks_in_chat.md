When a task ends with "and you need to do X, Y, Z in Unity," and X/Y/Z are short, also list them in the chat reply — don't only put them in a hand-off doc. The user is already reading chat; making them open and re-find a doc is friction.

**Why:** user said directly: "for the most part, if the stuff I need to do is small enough, can also list it in the chat so I don't have to find/review docs."

**How to apply:** if the user-facing TODO list is ≤ ~5 short bullets, inline them in the chat response. The hand-off doc is still fine to keep as a more detailed reference, but the chat should be self-sufficient for trivially short next steps.
